# Lexium Landing Page

Local replica of the Lexium landing page.

## Preview

```bash
npm start
```

Open `http://127.0.0.1:4173`.

## Deploy

This project is a static site. Vercel can deploy it directly from the repository root.

## Operational Inbox

All investor enquiries, law firm applications, legal correspondence, and press enquiries are routed to `contact@lexium.finance`. This inbox should be live and actively monitored from day one.
